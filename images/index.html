Recent Work

Normalized Differenced Vegetation Index on Colorado Burial Preserve

Created as a final project for a remote sensing course utilized NAIP composite imagery sourced through Google Earth Engine. The Normalized Difference Vegetation Index (NDVI) was calculated and analyzed using ArcGIS Pro to assess vegetation health and land cover characteristics.

Point Track Elevation Comparison

Completed as part of a GPS GIS course at Colorado Mountain College, this project compares point track elevation data along the Sage Creek Trail in Hayden, Colorado, using two data collection methods: the Avenza Maps mobile app and the professional-grade Bad Elf Flex Mini GPS device. The analysis evaluates the accuracy and reliability of consumer versus professional GPS tools for elevation tracking.

GPS Point Features: Dry Creek Parkscape

For a GPS GIS course at Colorado Mountain College, I created a detailed map and web map of Dry Creek Park in Hayden, Colorado, using GPS points collected with ArcGIS Field Maps. The project involved domain creation, feature class setup, custom symbology, and data collection timed with optimal satellite positioning using the Trimble GNSS Planning tool.

Normalized Differenced Vegetation Index of the Morgan Creek Fire in Routt County, CO

NDVI comparison maps created using pre-fire and post-fire Landsat 9 imagery from USGS EarthExplorer, illustrating changes in vegetation health before and after the fire.

Aggregated Whale Activity 2005-2008

This project involved aggregating whale location data collected between 2005 and 2008 using GPS and GIS techniques. Using ArcGIS Pro, I compiled and visualized spatial data to analyze whale movement patterns over time.

Potential Beaver Habitat in Sarvis Creek Wilderness

Created as a final project for Intermediate GIS at Colorado Mountain College. Areas that fit certain criteria for potential beaver habitat in Sarvis Creek Wilderness, including vegetation, slope, and proximity to a watersource.

Flat Top Wilderness Area Watershed Boundraies

Created a watershed boundary for the Flat Tops Wilderness area using raster-based hydrologic modeling in ArcGIS Pro. Utilized the Fill, Flow Direction, Sink, Con, Stream Link, and Watershed tools to process elevation data and accurately define watershed extents from raster imagery.
